#  RSM Telegram theme

Telegram themes for RSM pack

![Screenshot RSM](preview-rsm.png)

# Dependencies

- wget
- zip
- xdg-utils

# Download and Build

## RSM Telegram Theme:
```
wget -qO- https://raw.githubusercontent.com/Jacobet/RSM/master/extra/telegram/install-rsm-ttheme.sh | sh

```

# Install
Drag-n-drop **.tdesktop-theme** to Telegram
